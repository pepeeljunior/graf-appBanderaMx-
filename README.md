
Genera el código para una interfaz web interactiva de la "Bandera de México" dividida en 3 archivos separados: **index.html**, **style.css** y **script.js**.

**Requisitos de la Interfaz (HTML/CSS):**

1.  Debe mostrarse dentro de una tarjeta (card) centrada en la pantalla con el título "Bandera de México".
    
2.  La bandera debe dividirse en 3 columnas verticales alineadas horizontalmente:
    
    -   **Columna Izquierda (Verde):** Debe tener el texto "Esperanza", un input tipo `range` (slider), un texto mostrando el código Hexadecimal (inicial #009900) y un `div` cuadrado de color verde.
        
    -   **Columna Central (Blanca):** Debe tener el texto "Unidad", el texto "#FFFFFF" y un `div` cuadrado de color blanco. **Dentro de este cuadro blanco, inserta una etiqueta `<img>` con el Escudo Nacional de México** (usa una URL pública de Wikimedia), centrado vertical y horizontalmente.
        
    -   **Columna Derecha (Roja):** Debe tener el texto "Sangre de nuestros héroes nacionales", un input tipo `range`, un texto mostrando el código Hexadecimal (inicial #9D0000) y un `div` cuadrado de color rojo.
        
3.  **Pie de página:** Debajo de las columnas, pero dentro de la tarjeta, agrega un párrafo con el siguiente texto exacto: "Alan Alessandro Gonzalez Arroyo - 23201260 , Jose Luis Gomez Cruz - 23200841".
    

**Funcionalidad (JavaScript):**

1.  El slider de la izquierda debe modificar dinámicamente la intensidad del canal **Verde** (G) del recuadro verde y actualizar su texto Hexadecimal en tiempo real.
    
2.  El slider de la derecha debe modificar dinámicamente la intensidad del canal **Rojo** (R) del recuadro rojo y actualizar su texto Hexadecimal en tiempo real.
    
3.  El código debe ser limpio y moderno.
